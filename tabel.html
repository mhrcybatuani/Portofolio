Gambar tersebut menampilkan sebuah kode HTML yang digunakan untuk membuat tabel data mahasiswa. Bagian kiri menunjukkan struktur file proyek web, di mana kode HTML tersebut berada. Bagian kanan menampilkan hasil dari kode tersebut saat dijalankan pada web browser, yaitu sebuah tabel yang berisi data-data mahasiswa seperti nomor induk mahasiswa (NIM), nama, nilai IPK setiap semester, jurusan, dan fakultas. Kode HTML yang digunakan terdiri dari beberapa tag seperti table, tr, dan td yang berfungsi untuk mendefinisikan struktur tabel, baris, dan kolom.
